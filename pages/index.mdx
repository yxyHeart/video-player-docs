## 💥What is HPlayer?

- 一款由React编写的H5播放器,内置支持对mp4文件格式的流式播放;

- 同时如果借助例如dash.js,hls.js,flv.js,可以实现轻松适配任何流媒体协议例如MPEG-DASH,HLS,FLV等;

- 不仅如此,该播放器兼容PC端(macos && windows)和移动端(ios && android);

- 同时实现响应式设计，在不同的尺寸下组件会呈现不同的布局;

- 用户可以使用内部提供的Event API 来监听浏览器的各种事件、 Action API向播放器发出一些指令;


## Why choose it?

### ✅ 已完成

- React + typescript编写, 具有完善的类型系统

- 响应式设计, 内部时刻监听播放器尺寸的变化从而调整布局和样式

- 多端兼容，通过检测当前执行环境,对于移动端和PC端具有不同的布局和事件

- 适配了 mp4(手动解析), hls(调用hls.js) 流媒体播放协议
